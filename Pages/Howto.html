How to use this search bar?

This search bar will help you filter the contents in this website.

You may have noticed that there are codes attached to each item when you're searching for something. So what does BM, BK, TX, AU, AT, LY stands for?

BM: stands for bookmarks, my personal, useful links

BK: stands for books

TX: stands for online text ebooks

AU: stands for authors

AT: stands for articles

LY: stands for lyrics

These codes were added to help a user filter the wide-range of pages. So if you want the results to be only for books, you should enter:

BK: + the Title of the book

BK: El Filibusterismo

Additionaly, when searching for TX: files, you can also add - main page after the title if you want to go directly to the main page instead of the other pages.

That's all I have for you in this short tutorial. Have a great time navigating this website. :)
